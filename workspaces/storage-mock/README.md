# モック オブジェクトストレージ

このディレクトリ内に `public` ディレクトリを作成し、その中にサンプルの `.m3u8`, `.ts` ファイル一式を配置してください。

サンプルのダウンロード元： https://drive.google.com/open?id=1Q6MPg-4O73jYaX-_y8C8GVmFfCNPhnnY

## Docker コンテナの起動方法

```bash
docker-compose up -d
```

## Docker コンテナの停止

```bash
docker stop <CONTAINER ID>
```

## Docker コンテナの削除

```bash
docker rm <CONTAINER ID>
```
